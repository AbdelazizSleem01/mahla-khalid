# مميزات الموقع - مؤسسة مهله خالد للتجارة

## المواصفات العامة
- **الاسم**: موقع مؤسسة مهله خالد للتجارة
- **الوصف**: موقع متخصص في بيع البطاقات والكروت مسبقة الدفع
- **التقنيات الرئيسية**: Next.js 16, React 19, TypeScript, Tailwind CSS
- **التصميم**: DaisyUI مع دعم الوضع الفاتح والداكن
- **اللغة**: العربية (RTL)
- **الميزات**: تفاعلية عالية مع التحريكات

## الصفحات الرئيسية

### 1. الصفحة الرئيسية (/)
- قسم البطل (Hero) مع مقدمة الشركة
- قسم الخدمات مع عرض تفصيلي لكل خدمة
- قسم الفروع مع خرائط تفاعلية
- قسم من نحن مع الإحصائيات
- قسم التواصل (واتس اب، هاتف، سناب شات)
- قسم أوقات العمل

### 2. صفحة خدماتنا (/ourServices)
- عرض الخدمات بالتفصيل
- تصميم متجاوب

### 3. الصفحة الإدارية (/admin-1432-secret)
- لوحة تحكم شاملة
- إحصائيات وتحليلات
- تعديل كلمة المرور

### 4. صفحة 404 (/not-found)
- تصميم جميل مع أيقونات تعبيرية
- أزرار العودة للرئيسية والخلف

## الميزات التقنية

### الواجهة الأمامية (Frontend)
- **ربط تفاعلي**: Tab navigation للمعلومات المختلفة
- **تتبع الزوار**: Visitor tracking لإحصائيات الزيارات
- **دعم تعدد اللغات**: يدعم العربية بالكامل
- **تصميم متجاوب**: يعمل على جميع الأجهزة
- **وضع داكن/فاتح**: قابلية تغيير المظهر
- **تحريكات عنصرية**: Framer Motion للتفاعل
- **تحميل البيانات**: من ملف JSON محلي
- **صور تفاعلية**: إدارة الصور والوسائط
- **أكواد QR**: لرقم الحساب البنكي

### التحليلات والإحصائيات
- **الزيارات الإجمالية**: تتبع عدد الزوار
- **النقرات الإجمالية**: تتبع النقرات على الروابط
- **نقرات التبويبات**: تحليل التبويبات الأكثر استخداماً
- **أفضل الدول**: الدول الأكثر زيارة للموقع
- **الزيارات الحديثة**: قائمة بالزيارات الأخيرة مع التفاصيل
- **تحديث تلقائي**: تحديث البيانات كل 30 ثانية
- **تصدير البيانات**: تقارير مفصلة

### النظام الإداري
- **تسجيل الدخول**: نظام أمان مع رموز مرور
- **تحديث كلمة المرور**: تغيير كلمة المرور بأمان
- **إدارة المحتوى**: واجهة لإدارة البيانات
- **الجلسات المؤمنة**: حماية الجلسات الإدارية

### تبويبات المعلومات
- **المعلومات العامة**: بيانات الشركة الأساسية
- **شروط البيع**: الشروط والأحكام
- **أوقات الدوام**: مواعيد العمل اليومية
- **التقسيط**: معلومات الدفع المريح
- **العنوان**: معلومات الموقع
- **الموقع**: خرائط تفاعلية
- **البنوك**: معلومات الدفع البنكي
- **التواصل**: بيانات التواصل

### التكاملات
- **واتس اب**: تكامل مباشر
- **خرائط جوجل**: خرائط تفاعلية للفروع
- **سنس شات**: حساب رسمي لرؤية المزيد
- **الاتصال الهاتفي**: أزرار اتصال مدمجة

### قاعدة البيانات
- **MongoDB**: تخزين البيانات المحلية
- **إدارة SSL**: اتصالات آمنة
- **إلمام بقاعدة البيانات**: بنية محسنة للأداء

### الأداء والأمان
- **الحفاظ على الأداء**: تحسين الأعمال المثقلة
- **التصفح الآمن**: تدابير أمان قوية
- **تحسين الصور**: ضغط وتحسين تلقائي
- **تحميل تدريجي**: تحميل حسب الحاجة

### السيرفر (Backend)
- **API routes**: نقاط نهاية RESTful
- **معالجة مشفرة**: تشفير كلمات المرور
- **إدارة الجلسات**: رموز JWT آمنة
- **تحليلات متطورة**: استعلامات MongoDB متقدمة

### تجريب الميزات
- **إخطارات**: إشعارات عديمة الثبات للتفاعل
- **نماذج تفاعلية**: إدخالات مدعومة بتحقق
- **معالجة الأخطاء**: رسائل خطأ واضحة
- **وضع التطوير**: أدوات مساعدة المطورين
